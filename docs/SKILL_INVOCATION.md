# Skill Invocation

Short version:

1. session-start bootstrap teaches agents skill discipline
2. agent checks registry for available skills
3. process skills come before implementation skills
4. skill should be invoked before ad hoc exploration when applicable

## Primary Rules

- use skill tool, do not rely on memory
- check process skills first
- if debugging, planning, verification, or brainstorming applies, invoke that skill first
